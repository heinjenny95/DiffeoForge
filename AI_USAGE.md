# AI usage and accountability

AI assistance is not required to install or run DiffeoForge.

Generative AI tools, including OpenAI Codex, may be used during development for
code scaffolding, documentation drafts, test generation, and review support.
Material use must be disclosed in the relevant pull request or release notes.

Human contributors remain responsible for:

- defining scientific requirements and interpreting the methods;
- reviewing every accepted change;
- verifying licenses, references, tests, and numerical evidence;
- deciding authorship and contributor credit;
- reporting AI use according to the target journal's current policy;
- all claims made in documentation, releases, and publications.

AI-generated output is not treated as a scientific reference or validation
result. Numerical methods must be supported by primary literature, explicit
derivations where needed, tests, and controlled comparisons.

## Initial disclosure

The initial public project scaffold, architecture drafts, configuration schema,
tests, and documentation were developed with Codex assistance under the
direction of Jenny Hein. They are subject to human review through the public
